# 云开壁纸新标签页（Next）
* 本插件是云开新标签页（Legacy）的重构版本，新版移除了多余功能，优化了性能和用户体验。

## 简介
* 在浏览器新标签页随机展示来自 [Unsplash](https://unsplash.com) 的图片。
* 支持语言：简体中文。
* 所需权限：storage。
* 声明：图片获取方式为随机推送，内容不代表本人的观点。本插件已将 API 过滤等级设为最高，如仍旧出现不合适的内容，可前往 Unsplash 图片主页反馈。
